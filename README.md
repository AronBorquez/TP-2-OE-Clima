# Escenario seleccionado
Análisis de Datos Climáticos

# Descripción del proyecto
Este proyecto tiene como objetivo analizar datos climáticos históricos utilizando Python, Git, GitHub y Jira como herramientas de gestión colaborativa y control de versiones.

# Dataset utilizado
Dataset de temperaturas globales en formato CSV obtenido desde DataHub.

# Estructura del repositorio
/datos → contiene el dataset utilizado
/scripts → contiene los scripts de análisis
/resultados → contiene gráficos y resultados generados

# Tecnologías utilizadas
Python
Google Colab
Git
GitHub
Jira

# Ejecución del proyecto
1. Clonar el repositorio
2. Ejecutar el script ubicado en /scripts
3. Ver los resultados generados en /resultados
